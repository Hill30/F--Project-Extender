1. General

Installing F# Project Extender provides tools for better organization of files in F# projects by allowing project subdirectories and separating file management in Solution Explorer from controlling compilation order.

To enable Project Extender right click on your FSharp project file in Solution Explorer and click 'Enable F# Project Extender'. After that in Properties window of your project you may see additional sub panel 'Compilation Order' which allows you to better organize F# files inside your project.
